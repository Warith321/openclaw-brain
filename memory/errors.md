# errors.md - 错误学习日志

记录失败案例，避免重蹈覆辙。

## 格式
```
### YYYY-MM-DD: 错误标题
- **现象**: 发生了什么
- **原因**: 为什么出错
- **修复**: 怎么解决的
- **教训**: 以后怎么避免
```

---

### 2026-03-04: PowerShell 不支持 CMD 语法
- **现象**: `rmdir /s /q` 报错
- **原因**: PowerShell 和 CMD 语法不同
- **修复**: 改用 `Remove-Item -Recurse -Force`
- **教训**: Windows PowerShell 环境下给对应的 PS 语法，不要给 CMD 命令

### 2026-03-04: 飞书配对码有时效性
- **现象**: 配对码发来时已过期，approve 失败
- **原因**: 配对请求有时间窗口
- **修复**: 让用户重新触发，立即批准
- **教训**: 配对码要第一时间处理，不能等

### 2026-03-04: feishu 插件重复导致冲突
- **现象**: 启动时 duplicate plugin id 警告，程序卡住
- **原因**: 同时开启了内置 feishu 和扩展 feishu
- **修复**: 关掉其中一个，删除扩展目录
- **教训**: 同名插件只能保留一个
